詳細説明（Detailed description）

PubMed2EndNoteは、PubMedの論文情報をワンクリックでEndNote対応の引用に変換するChrome拡張機能です。 【使い方】 1. PubMed（pubmed.ncbi.nlm.nih.gov）で論文ページを開く 2. ページ右上に表示される青いボタンをクリック 3. Microsoft Wordに Ctrl+V で貼り付け 4. EndNoteタブの「Update Citations and Bibliography」で引用と文献リストが自動整形されます 【特徴】 ・著者名・雑誌名・巻号ページ・DOI・Abstract・MeSHタームなどの書誌情報を自動取得 ・EndNote（Cite While You Write）がそのまま認識できる引用フィールドとして貼り付け可能 ・貼り付け箇所は黄色ハイライトで目立つように表示 ・ウムラウトなどの特殊文字にも対応 ・書誌情報は米国国立医学図書館（NCBI）の公式APIから取得 【必要な環境】 ・Microsoft Word ＋ EndNote（Cite While You Write） ・初回利用時にメールアドレスの設定が必要です（NCBI APIの利用ガイドラインに従い、APIリクエストの連絡先として送信されるものです） --- PubMed2EndNote converts PubMed article metadata into an EndNote-compatible citation with one click. Click the blue button on any PubMed article page, then paste into Microsoft Word (Ctrl+V). EndNote's Cite While You Write recognizes the pasted citation as a proper citation field — just run "Update Citations and Bibliography" to format it.

3. プライバシー（「プライバシーへの取り組み」タブ）

単一用途の説明（Single purpose description）

This extension converts PubMed article metadata into an EndNote-compatible citation and copies it to the clipboard, so that it can be pasted into Microsoft Word as a citation field recognized by EndNote (Cite While You Write).

権限の使用理由（Permission justifications）

storage
Stores the user's preferences entered on the options page: an optional email address and citation formatting settings (font, size, highlight). The email address, if provided, is appended to NCBI E-utilities API requests as a contact parameter, as recommended by NCBI's API usage guidelines. Nothing is used for any other purpose.
clipboardWrite
Writes the formatted EndNote citation to the clipboard when the user clicks the copy button on a PubMed article page. This is the core function of the extension.
ホスト権限
（content script:
pubmed.ncbi.nlm.nih.gov）
Injects a copy button on PubMed article pages (pubmed.ncbi.nlm.nih.gov). This is the sole surface of the extension; it does not read or modify page content beyond adding this button.

# Privacy Policy — PubMed2EndNote Last updated: July 4, 2026 ## Data collected PubMed2EndNote lets the user optionally enter an email address on its options page. If provided, this email address is: - stored locally in the browser (chrome.storage.sync), and - appended as a contact parameter to requests sent to the NCBI E-utilities API (https://eutils.ncbi.nlm.nih.gov/), as recommended by NCBI's API usage guidelines. The extension is fully functional without providing an email address. The email address is NOT sent to the developer or to any third party other than NCBI, and is not used for tracking, advertising, or any other purpose. ## Data NOT collected The extension does not collect browsing history, page content, or any analytics. 5. 審査に関するメモ

  • 審査期間は通常1〜3営業日程度（初回はやや長引くことあり）
  • 権限は storage と clipboardWrite のみ（どちらもインストール時の警告なし）。ホスト関連はPubMedページへのcontent script注入のみで、審査上もっとも軽量な部類
  • Manifest V3準拠、リモートコードなし、難読化なし — 指摘されやすいポイントはすべてクリア
  • リジェクトされた場合は理由がメールで届くので、そのまま転送してもらえれば対応します
