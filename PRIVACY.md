# Privacy Policy — PubMed2EndNote

Last updated: July 4, 2026

## Data collected
PubMed2EndNote asks the user to enter an email address on its options page.
This email address is:
- stored locally in the browser (chrome.storage.sync), and
- appended as a contact parameter to requests sent to the NCBI E-utilities API
  (https://eutils.ncbi.nlm.nih.gov/), in compliance with NCBI's API usage policy.

The email address is NOT sent to the developer or to any third party other than
NCBI, and is not used for tracking, advertising, or any other purpose.

## Data NOT collected
The extension does not collect browsing history, page content, or any analytics.
All citation processing happens locally in the browser.
